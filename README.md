# 🩺 Orthopedic Clinical Atlas

אטלס קליני אורתופדי בעברית - הכלי המקצועי המקיף ביותר בעברית לאבחון אורתופדי קליני.

## ✨ פיצ'רים

- 🩺 **91 פתולוגיות** עם 470 בדיקות קליניות
- 🎯 **23 קלאסטרים אבחנתיים** עם חישוב Bayesian
- 📊 **11 שאלוני Outcome Measures** (NDI, ODI, QuickDASH, KOOS-PS, HOOS-PS, FAAM, VAS, PSFS, GROC, FABQ, TSK)
- 🚨 **19 דגלים אדומים** ב-6 קטגוריות
- 🧠 **בדיקה נוירולוגית מקיפה** (Myotomes, Dermatomes, Reflexes, UMN, Neural Tension)
- 🦴 **אטלס אנטומיה ו-ROM** ל-9 אזורים
- 👥 **כרטיסי מטופלים** עם ייצוא PDF/JSON
- 🗺️ **מפת Dermatomes ויזואלית**
- 📚 **מאגר 80 שרירים** עם עצבוב

## 🚀 הרצה מקומית

```bash
npm install
npm run dev
```

## 📦 בנייה ל-Production

```bash
npm run build
```

הפרויקט מוכן לפריסה ב-Vercel, Netlify, או GitHub Pages.
